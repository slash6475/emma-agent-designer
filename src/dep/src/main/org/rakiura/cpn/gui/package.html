Provides a GUI to draw, show, edit, simulate and so on coloured Petri nets.

The main class is {@link org.rakiura.cpn.gui.NetToolboxApplication NetToolboxApplication.java}, which has a main method to start the Toolbox application.

The drawings are based upon the {@link org.rakiura.draw} package.

Every net can have multiple layouts. Each layout can only be used for the net it was created for, and only if the net has not changed.
You can load a net, a netdrawing or a layout. The missing elements will be regenerated.
